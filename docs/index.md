## Free vs Pro at a glance

| Capability                                                       | Free | <span class="doc-pro-pill">Pro</span> |
| ---                                                              | --- | --- |
| Text watermark (font picker, color, opacity, size 8–72 px)       | ✅  | ✅  |
| Image watermark (PNG / SVG via Imagick, scale, opacity, position)| ✅  | ✅  |
| 9-point positioning grid + X/Y offset (px or %)                  | ✅  | ✅  |
| Watermark on upload (automatic)                                  | ✅  | ✅  |
| Watermark on REST media metadata pipeline                        | ✅  | ✅  |
| Originals backup (`uploads/ulwm-backup/YYYY/MM/`)                | ✅  | ✅  |
| Right-click + drag-and-drop protection                           | ✅  | ✅  |
| Per-subsize control (apply to thumbnails, full, large…)          | ✅  | ✅  |
| Conditional rules: post type + image size                        | ✅  | ✅  |
| Bulk media-library actions (apply / remove)                      | ✅  | ✅  |
| Multiple stacked watermarks per image                            | ✅  | ✅  |
| GD or Imagick auto-detection                                     | ✅  | ✅  |
| **One** watermark template (free limit)                          | ✅  | —   |
